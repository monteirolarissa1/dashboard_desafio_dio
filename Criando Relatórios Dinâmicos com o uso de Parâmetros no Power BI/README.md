# Relatório Gerencial de Vendas - Data Analytics

## Visão Geral
Este projeto foi desenvolvido tendo como objetivo o uso da função parâmetro em visuais.

---

**OBS: este projeto é resposta de um desafio proposto pelo Bootcamp de Analise de dados, parceria entre a plataforma Dio e Klabin.**

O repositório com o desafio original pode ser encontrado em: [Desafio Power BI Curso - Juliana Mascarenhas](https://github.com/julianazanelatto/power_bi_analyst/tree/main/M%C3%B3dulo%204/Desafios%20de%20Projeto)


---

## Estrutura do Projeto

O resultado do desafio foi inserido na aba "Parâmetro - Segmento" do relatório, sendo que as outras abas estão ocultas para dar foco na atividade proposta no desafio.

Através desta aba, é possível ter a visualização em gráfico de barras da soma de lucro por Segmento, País, Produto e Mês, de acordo com o parâmetro selecionado.
