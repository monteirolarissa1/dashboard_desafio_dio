# Relatório Gerencial de Vendas - Data Analytics

## Visão Geral
Este projeto foi desenvolvido como parte do **Desafio de Projeto - Atualizando Relatório Financeiro com Foco na Experiência do Usuário**, com o objetivo de aprimorar um relatório anterior, aplicando princípios de design, navegabilidade e segmentação de dados para uma experiência de usuário mais intuitiva e profissional.

O projeto foi desenvolvido no **Power BI** e utiliza boas práticas de **design de dashboards corporativos**.

---

## Objetivo do Desafio
Atualizar o relatório de vendas existente com foco na **experiência do usuário (UX)**, aplicando conceitos como:

- **Posicionamento visual** de elementos (hierarquia e alinhamento);
- **Contraste** de cores para leitura e acessibilidade;
- **Proporção áurea** para equilíbrio estético;
- **Segmentação de dados** para navegação intuitiva;
- **Padronização de alinhamento seguindo modelo de Fibonacci**;
- **Criação de botões e menus interativos**.

O desafio visa unir análise de dados e storytelling visual, promovendo uma entrega mais clara e estratégica para tomadores de decisão.

## Estrutura do Projeto

O relatório é composto por **4 páginas principais**:

1. **Visão Geral de Vendas**
   - Indicadores principais (faturamento, lucro, margem)
   - Gráfico de área modificado para melhor leitura temporal
   - Matriz de descrição de vendas (por região, produto ou categoria)

2. **Visão inicial de Lucro**
   - Filtros interativos e segmentações dinâmicas
   - Tabelas e visuais detalhados de desempenho
   - Interação com botões de navegação
   - Imdicadores macro relacionados a lucro.

3. **Lucro vs Vendas**
   - Comparativo entre lucro e vendas
   - Graficos por regionais
   - Grafico de funil de vendads e lucros.

4. **Produto**
   -Indicadores por produto, sobretudo relacionados aos 3 produtos com maior saida e maior receita.
